# Academic Paper Formatter (GitHub Actions)

Kullanım:
1. Repo içinde "uploads/" klasörü oluşturun (aşağıdaki adımla).
2. uploads/ klasörüne .docx dosyanızı yükleyin (Add file > Upload files).
3. GitHub Actions otomatik olarak çalışacak.
4. Actions > ilgili run > Artifacts bölümünden "processed-docx" artifact'ini indirin.
   İçinde dosyaniz_formatted.docx olacaktır.

Notlar:
- Sadece .docx desteklenir.
- Basit stil değişiklikleri uygulanır (varsayılan font, punto, kenar boşlukları, Heading 1 için örnek).
